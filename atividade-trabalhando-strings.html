Exercício 1

Crie um programa que concatene dois nomes e retorne uma frase de saudação. Entrada: nome1 = "João", nome2 = "Silva" Saída: "Olá João Silva!"


                        let nome1 ="João";
                        let nome2 = "Silva";

                        console.log(`Olá, ${nome1} ${nome2}!`);

Exercício 2

Crie um programa que adicione uma string ao final de outra string e retorne a string resultante. Entrada: string1 = "hello", string2 = "world" Saída: "helloworld"


                        let string1 = "hello";
                        let string2 = "world";

                        console.log (`${string1}${string2}`);

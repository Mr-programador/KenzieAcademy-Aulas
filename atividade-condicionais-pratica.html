Exercício 2

Faça um algoritmo para receber um número qualquer e informar na tela se é par ou ímpar.


                        let numero = 10;
                        if (numero % 2 === 0) {
                            console.log("O número é par.");
                        } else {
                            console.log("O número é ímpar.");
                        }
